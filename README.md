# Hello Android Project

This is my first Kotlin-based Android application created in Android Studio.  
The app simply displays "Hello World" on the screen, serving as a starting point for learning Android development and GitHub integration.

## Purpose

The purpose of creating this software is to strengthen my programming skills in Kotlin, learn how to manage game logic, and practice integrating projects with GitHub.  
It also serves as a foundation for exploring cloud integration later (e.g., storing high scores with Firebase).
